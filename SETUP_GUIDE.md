# সেটআপ গাইড

নবীনদের জন্য এই প্রকল্প সেটআপ করার জন্য এই ধাপগুলো অনুসরণ করুন।  

## ১. প্রয়োজনীয়তা  
 - [Node.js](https://nodejs.org/)  
 - [Git](https://git-scm.com/)  

## ২. প্রকল্প ক্লোন করা  
প্রকল্পটি ক্লোন করতে নিচের কমান্ডটি ব্যবহার করুন:  
```bash  
git clone https://github.com/abidabdulrahman23-crypto/BD-FIRE-ZONE-.git  
```  

## ৩. ডিপেন্ডেন্সি ইনস্টল করা  
প্রকল্পের ডিরেক্টরিতে প্রবেশ করুন:  
```bash  
cd BD-FIRE-ZONE-  
```  
তারপর, ডিপেন্ডেন্সি ইনস্টল করতে:  
```bash  
npm install  
```  

## ৪. প্রকল্প চালানো  
প্রকল্পটি চালাতে নিচের কমান্ডটি ব্যবহার করুন:  
```bash  
npm start  
```  

## ৫. ব্রাউজারে খুলুন  
আপনার ব্রাউজারে `http://localhost:3000` খুলুন।
